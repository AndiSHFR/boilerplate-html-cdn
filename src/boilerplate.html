Html / JavaScript Boilerplate
About this template

What's it about

This is a minimal html template as a starting point for html/javascript coding.

There are no local css- or javascript-files. All external files are pulled in from content delivery networks (CDN).

What do you get?

The following frameworks are included in this template:

  • Google Font - Varela Round
  • jQuery
  • Bootstrap
  • Knockout
  • Redux

How to use it?

Start coding css/html/Javascript in this file.

Remove content as you wish (nav bar, panel, ...).

Happy coding!
